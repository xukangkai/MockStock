# A股交易Agent拟人化图标设计规范

## 1. 设计理念

### 1.1 设计目标
为7个交易专家Agent设计拟人化视觉形象，在保留核心功能识别特征的前提下赋予其人物性格，增强用户对多智能体协作系统的认知感和亲和力。

### 1.2 核心原则
- **功能可识别**：每个Agent图标通过特征道具直接映射其核心职责
- **风格统一性**：同一系列内造型比例、色彩逻辑、视觉语言保持一致
- **UI适配性**：与现有深色系交易终端UI协调，提供3种风格供不同场景选择
- **可扩展性**：矢量SVG格式，支持任意缩放和后续衍生设计

## 3. 三种设计风格方案

### 方案A：Q版卡通风 (styleA_cartoon)

**适用场景**：用户引导页、新手教程、移动端APP、社交分享图

**视觉特征**：
- 大头Q版比例，头身比约 2:1
- 圆润线条，大腮红，可爱表情（弯月眼/豆豆眼）
- 柔和肤色渐变 (#ffe8d6 → #ffd4b8)
- 道具倾斜摆放，姿态活泼
- 背景：双层圆形渐变晕染

**色彩标准**：
- 肤色：径向渐变 #ffe8d6 → #ffd4b8
- 腮红：#fca5a5，55% 透明度
- 线条：#1e293b 深灰
- 背景渐变：主色 → 辅色，14%/8% 两层透明度

---

### 方案B：专业商务风 (styleB_professional)

**适用场景**：PC端专业交易终端、机构版界面、正式汇报材料

**视觉特征**：
- 标准人像比例，头部略大于肩部
- 深色商务发型 + 西装外套 + 衬衫领 + 领带
- 部分角色佩戴眼镜（记忆/市场分析）
- 道具精致规整，水平摆放
- 背景：双层圆角矩形渐变边框

**色彩标准**：
- 肤色：径向渐变 #fde8d2 → #f1c9a5
- 西装：主色 100%，翻领 35% 透明度叠加
- 领带：辅色 100%
- 发型：#1e293b 深黑 + #334155 高光层
- 线条：#1e293b / #334155

---

### 方案C：赛博科技风 ⭐ 默认推荐 (styleC_cyberpunk)

**适用场景**：当前项目深色UI、量化交易系统、科技感展示页

**视觉特征**：
- 机械头盔式发型 + 电路纹路装饰
- 发光霓虹眼睛（GaussianBlur 滤镜）
- 机械身体 + 胸口能量核心（带呼吸光晕）
- 脸部两侧电路节点装饰
- 四角科技感装饰边框
- 道具：机械风 + 发光描边

**色彩标准**：
- 背景：深色渐变 #0b1220 → #020617
- 肤色：线性渐变 #e2c4ad → #c79b7a
- 霓虹描边：主色 (primary) 带 glow 滤镜
- 能量节点：辅色 (secondary) 带 glow 滤镜
- 外边框：主色 50% 透明度 + 辅色 20% 透明度 双层
- 滤镜：feGaussianBlur stdDeviation="2" 发光效果

## 5. 应用指南

### 5.1 尺寸规范

| 使用场景 | 推荐尺寸 | 格式 | 示例 |
|---------|---------|------|------|
| 导航栏/标签图标 | 24-32px | SVG | 侧边栏菜单 |
| Agent 卡片头像 | 42-56px | SVG | 专家团队卡片 |
| 详情页大图 | 128-256px | SVG/PNG@2x | Agent 介绍页 |
| 登录页/引导页 | 256-512px | PNG 256 | 新手引导 |
| Favicon/应用图标 | 32-64px | PNG 64 | 浏览器标签 |

> 💡 **优先使用SVG**：浏览器原生支持，任意缩放不模糊，文件体积极小（单图标3-6KB）

### 5.2 HTML集成示例

#### SVG 直接引用（推荐）
```html
<!-- Agent 卡片头像：42×42 赛博风 -->
<img src="/static/agents/styleC_cyberpunk/svg/memory.svg"
     alt="记忆检索"
     style="width:42px;height:42px;border-radius:50%;object-fit:cover;">
```

#### 总指挥大头像
```html
<!-- 总指挥卡片：56×56 -->
<img src="/static/agents/styleC_cyberpunk/svg/commander.svg"
     alt="总指挥"
     style="width:56px;height:56px;border-radius:50%;box-shadow:0 0 20px rgba(99,102,241,0.3);">
```

### 5.3 风格选择建议

| 用户群体 | 推荐风格 | 理由 |
|---------|---------|------|
| 专业交易者/机构用户 | 方案B 商务风 | 稳重、专业、信任 |
| 科技爱好者/量化用户 | 方案C 赛博风 ⭐ | 科技感、量化感、契合深色UI |
| 新手散户/大众用户 | 方案A Q版风 | 亲和、降低门槛、趣味性 |

### 5.4 状态叠加规则

Agent 运行状态（idle/thinking/done）通过边框/光晕叠加，不修改图标本身：
- **idle（等待）**：图标整体 45% 透明度
- **thinking（工作中）**：主色 1.5px 边框 + 呼吸光晕动画
- **done（已完成）**：辅色 1px 低透明度边框

## 6. 生成脚本使用

### 重新生成图标
```bash
cd /path/to/project
python3 generate_agent_icons.py
```

### 高清PNG渲染（需要系统cairo库）
```bash
# macOS:
brew install cairo
pip install cairosvg
python3 generate_agent_icons.py

# Ubuntu:
sudo apt-get install libcairo2
pip install cairosvg
```

### 修改Agent配色
编辑 `generate_agent_icons.py` 顶部 `AGENTS` 列表中的 `primary` / `secondary` 字段，重新运行脚本即可。
